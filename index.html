HTML

  • HTML stands for Hypertext Markup Language
  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • HTML is the structure for the webpage being created
  • HTML elements are usually composed of an opening tag and a closing tag represented with angle brackets ("<>")
  • elements that are nested inside other elements are called child elements
  • the element that holds all the children elements are known as the parent element
  • an element refers to the entire element from opening tag to closing tag
  • a tag refers only to what appears inside the angle brackets
